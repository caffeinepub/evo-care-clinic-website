# Evo Care Clinic Website

## Overview
A fully responsive dermatologist clinic website for Evo Care Clinic featuring Dr. Jeel Sejpal's services. The website uses a soft purple and white color theme matching the clinic's branding.

## Core Features

### Homepage
- Hero banner with clinic logo and tagline
- Key services overview: Skin, Hair, Laser, Ayurveda treatments
- Call-to-action for appointments
- Contact information display

### About Clinic
- Overview of Evo Care Clinic
- Dr. Jeel Sejpal's credentials and specializations
- Clinic mission and values

### Services
Individual service pages/sections for:
- PRP Hair Restoration
- Acne Care treatments
- Skin Brightening and Glow treatments
- Glutathione IV Drip therapy
- Advanced Skin Rejuvenation

Each service includes description, benefits, and booking information.

### Gallery
- Image gallery showcasing clinic graphics and treatment visuals
- Responsive grid layout for different screen sizes

### Doctors Profile
- Dr. Jeel Sejpal's detailed profile
- Contact information and specialization details
- Professional credentials

### Contact Form
Contact form with fields:
- Name
- Email address
- Phone number
- Message/inquiry

### Location by Continent
- Dropdown menu for continent selection (Asia, Europe, Africa, etc.)
- Display regional clinic information or placeholder content based on selection

### Footer
- Complete contact details
- Address: 204, Rk empire, Near Maudi Circle, 150ft Ring Road, Mavdi, Rajkot
- Social media links (placeholders)
- Quick navigation links

### Project Download
- Download functionality to package the entire project (frontend and backend code, including assets) into a ZIP file
- Direct download link provided to users
- ZIP file contains complete source code and all project assets
- Generate fresh ZIP file with updated source code upon request

## Technical Requirements

### Responsive Design
- Mobile-first approach
- Optimized for phones, tablets, and desktops
- Touch-friendly navigation and interactions

### Design Theme
- Primary colors: Soft purple and white
- Clean, professional medical aesthetic
- Consistent branding throughout

### Content Language
- All content in English
- Professional medical terminology where appropriate

## Data Storage
The backend stores:
- Contact form submissions
- Appointment requests
- Regional clinic information for continent-based display

- Include all frontend and backend files in the ZIP package
- Include all project assets and dependencies
- Provide direct download endpoint for the updated ZIP file
